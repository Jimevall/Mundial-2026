# ⚽ Mundial 2026 - Página Web Interactiva

¡Bienvenido a la página web oficial del fan sobre la **Copa Mundial FIFA 2026**!

## 🌍 Descripción

Esta es una página web interactiva desarrollada para proporcionar información completa y actualizada sobre el Mundial 2026, que se llevará a cabo en **Estados Unidos, México y Canadá**.

## ✨ Características

- **Diseño Responsivo**: Se adapta perfectamente a dispositivos móviles, tablets y desktop
- **Navegación Intuitiva**: Menú sticky con navegación suave entre secciones
- **Información Completa**:
  - 🏟️ Sedes principales y estadios
  - 📅 Calendario preliminar
  - ⚽ Equipos participantes (48 equipos)
  - 🎫 Detalles de todos los estadios
- **Formulario de Suscripción**: Recibe noticias del Mundial
- **Animaciones y Efectos**: Transiciones suaves y atractivas
- **Optimizado SEO**: Meta tags y estructura HTML semántica

## 📁 Estructura del Proyecto

```
mundial-2026/
├── index.html          # Archivo HTML principal
```

## 📊 Secciones Principales

### 1. Hero Banner
Bienvenida con banderas animadas de los tres países anfitriones.

### 2. Sedes Principales
Tarjetas interactivas con información sobre las sedes principales.

### 3. Estadios
Grid completo con los 12 estadios donde se jugará el torneo.

### 4. Calendario
Tabla detallada con las fases del torneo y fechas.

### 5. Equipos
Grid de 12 equipos principales con badges de clasificación.

### 6. Contacto
Formulario para suscribirse a las noticias del Mundial.
